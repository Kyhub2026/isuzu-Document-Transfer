📊 เชื่อมต่อ Google Sheets

วาง Google Apps Script Web App URL เพื่อให้ระบบบันทึกข้อมูลลง Google Sheets อัตโนมัติ

วิธีสร้าง:
1. เปิด Google Sheets ใหม่
2. ไปที่ Extensions → Apps Script
3. วาง Script ด้านล่างแล้ว Deploy → Web App
4. ตั้ง Execute as: Me, Who has access: Anyone
